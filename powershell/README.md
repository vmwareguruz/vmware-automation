# PowerShell Scripts

This folder contains PowerShell scripts for VMware automation.

Suggested structure:
- `reporting/` – inventory, capacity, health reports
- `remediation/` – configuration fixes, drift correction
- `vrops/` – Aria Operations (vROps) API and reporting scripts
- `esxi/` – host-level tasks
- `vsphere/` – vCenter/vSphere automation
- `utils/` – shared helper functions/modules
